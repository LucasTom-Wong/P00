## How to install and run our website:

### 1. Clone the repository in terminal:
```
git clone <ssh link>
```

### 2. Create a virtual enviroment:
```
python3 -m venv ~/virtualenvname
```

### 3. Install flask:
```
pip3 install flask
```

### 4. Navigate to the directory and run:
```
python3 app.py
```
